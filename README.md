### Function 功能  
Download all bounty items from chaos and unzip them into a separate folder  
从[chaos](https://chaos.projectdiscovery.io/)下载所有提供赏金的项目,解压到单独的文件夹  
### Usage 用法 
```python get_chaos.py```
